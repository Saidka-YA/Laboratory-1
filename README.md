# Laboratory-1
Лабораторная работа №1: Работа с Git и GitHub

Описание проекта: Данный проект создан в рамках выполнения лабораторной работы №1. Цель работы: изучение платформы GitHub и практическое использование Git для хранения, обновления и распространения исходного кода проекта.

Задание по варианту: Вариант 1 — Геометрические фигуры.

Реализованы функции для вычисления параметров следующих фигур (распределены между участниками группы): Прямоугольник: периметр, площадь, длина диагонали; Треугольник: периметр, площадь по формуле Герона, проверка на равнобедренность; Окружность: длина окружности, площадь круга, площадь кругового сектора; Трапеция: периметр, площадь, длина средней линии.

Команда проекта: Рожков Дмитрий — разработка модуля «Прямоугольник»; Матющенко Матвей — разработка модуля «Треугольник»; Ландин Дамир — разработка модуля «Трапеция»; Почоев Сайдулло — разработка модуля «Окружность»;

Используемые технологии >> Язык программирования: C++

Система контроля версий: Git

Платформа для хостинга репозиториев: GitHub
